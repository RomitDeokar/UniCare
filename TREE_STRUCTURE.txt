## 🎯 Key Directories Explained

### 📱 app/ - Main Application
The heart of your Next.js application using App Router:
- **page.tsx** - Routes and pages
- **layout.tsx** - Layouts and wrappers
- **api/** - Backend API endpoints
- **dashboard/** - Protected dashboard area

### 🎨 components/ - Reusable UI
React components used throughout the app:
- **ui/** - Base components (buttons, cards, inputs)
- **providers.tsx** - Context providers

### 🔧 lib/ - Utilities
Helper functions and configurations:
- **prisma.ts** - Database client singleton
- **utils.ts** - Common utilities

### 🗃️ prisma/ - Database Layer
ORM configuration and schema:
- **schema.prisma** - Database models and relationships
