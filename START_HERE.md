## 🚀 How to Run (5 Minutes!)

### Option 1: Automated Installation

```bash
cd unicare-hms
chmod +x install.sh
./install.sh
```

### Option 2: Manual Installation

```bash
# 1. Install dependencies (2 min)
npm install

# 2. Setup MySQL (1 min)
mysql -u root -p < hospital_ddl.sql
mysql -u root -p < hospital_dml.sql

# 3. Configure environment (1 min)
cp .env.example .env
# Edit .env with your MySQL password

# 4. Generate Prisma (30 sec)
npx prisma generate

# 5. Run! (30 sec)
npm run dev
```

### 🌐 Then Visit:
- **http://localhost:3000** - Landing page
- **http://localhost:3000/dashboard** - Dashboard
- **http://localhost:3000/dashboard/patients** - Patient management
- **http://localhost:3000/dashboard/emergency** - Emergency cases
